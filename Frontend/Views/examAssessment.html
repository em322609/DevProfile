Exam-Assessment Management and Portal System

The EAMPS solution is developed for managing a repository of questions and provide features for conducting examinations. User roles are defined for administrators, trainers, and associates (i.e. students). As an associate, they can take an exam, submit an exam, review answers, and view their scores. The trainers have the ability to schedule an exam, view scores by associate and at group level, and add/delete/edit questions. Administrators have additional features, such as creating and editing exam templates, and editing categories/topics/subjects. This application provides an AngularJS front-end for users to interact with available features, multiple web services that contain the business logic, and data access services to manage a database.


Roles & Responsibilities

  • Leveraged AngularJS into Bootstrap CSS and HTML to generate views for the application
  • Developed AngularJS services and factories to share data between controllers
  • Integrated UI.router to handle routing functionality in AngularJS for the front-end of the application
  • Integrated two-way data binding in AngularJS to share data between controllers.
  • Developed a Web API to send and receive user login information in JSON format
  • Database design and implementation completed using SQL and SQL Server Management Studio
  • Integrated Entity Framework to communicate with a SQL database
  • Followed AGILE development methodologies and practices using Pivotal Tracker

Environments & Technologies

ASP.NET Web API, AngularJS, SQL Server, JavaScript, C#, ADO.NET Entity Framework, HTML, CSS, Bootstrap, .NET, ASP.NEt
