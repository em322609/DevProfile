Booking and Ride Finder

Ride finder application provides features for riding sharing and taxi services. Users can search available rides by location and destination. They can also book a ride service, including entering pick-up and drop-off locations, and save their profiles. An interactive map provides driver locations, pick-up address, and routes requested. User interaction is done through a client-side application which consumed business logic web services, with access to a back-end database.


Roles & Responsibilities

  • Incorporated Bootstrap CSS into my HTML markups for styling and to create responsive web design
  • Integrated Web API RESTful Web Service to send and receive information in JSON format
  • Consumed my Web API with an ASP.NET MVC application
  • Back-end technologies included using ADO.NET and Entity Framework with SQL Server to store and manipulate data
  • Used C# to handle business logic within my application
